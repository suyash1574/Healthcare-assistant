MedlinePlus XML Files

MedlinePlus produces XML data sets that you are welcome to download and use. If you have questions about the
MedlinePlus XML files, please contact us [https://support.nim.nih.gov/support/create-case/?
category=medlineplus&from=https://medlineplus.gov/xml.html/] . For additional sources of MedlinePlus data in XML
format, visit our Web service [https://medlineplus.gov/about/developers/webservices/] page. If you are looking for
data from MedlinePlus Genetics, please see MedlinePlus Genetics Data Files & API
[https://medlineplus.gov/about/developers/geneticsdatafilesapi/] .

If you use data from MedlinePlus XML files or build an interface that utilizes the files, please indicate that the
information is from MedlinePlus.gov [https://medlineplus.gov/index.html] . Please see NLM’s API page
[https://eresources.nlm.nih.gov/nim_eresources/] for further guidance.

Health Topics

MedlinePlus publishes three types of health topic XML files daily (Tuesday-Saturday):
e MedlinePlus Health Topic XML
e MedlinePlus Compressed Health Topic XML
e MedlinePlus Health Topic Group XML

The six most recent files and their corresponding DTDs are linked at the bottom of this section.

The MedlinePlus health topic XML files contain records for all English [https://medlineplus.gov/all_healthtopics.html]
and Spanish [https://medlineplus.gov/spanish/all_healthtopics.html] health topics. Each health topic record includes

data elements associated with that topic. This associated data includes:

e basic metadata (health topic title, URL, language, date created, and ID),

vocabulary (MeSH, "Also called" terms, and see references),
e full summary,
* group membership,

¢ related health topics,

equivalent English or Spanish topics,

https://medlineplus.gov/xmI.html 1/4

e related content in other languages,

e primary NIH institute, and

¢ all site records (links) assigned to a health topic page, including their names, URLs, organizations, category
assignments, and standard descriptions.

These XML files allow you to download and use virtually all text and links that appear on MedlinePlus health topic
pages. For complete details on all elements and attributes in the MedlinePlus health topic XML, see the MedlinePlus
XML file description [https://medlineplus.gov/xmldescription.html] .

The MedlinePlus Compressed Health Topic XML contains the same information as the MedlinePlus Health Topic
XML, but it is posted as a .zip file for easier downloading.

The MedlinePlus health topic group XML files contain information on all English
[https://medlineplus.gov/healthtopics.html] and Spanish [https://medlineplus.gov/spanish/healthtopics.html] topic
groups.
